Bilig WorkPaper is an open-source TypeScript runtime for using spreadsheet-style formulas inside Node.js services and AI agent tools. It lets developers edit workbook cells through an API, recalculate formulas, read values back, and persist the workbook as JSON without driving Excel or Google Sheets through a UI.

The project is useful for pricing rules, payout checks, import validation, quote calculators, and agent workflows where spreadsheet logic needs to stay reviewable but also run in backend code. Its MCP server exposes workbook operations such as listing sheets, reading ranges, setting cell contents, validating formulas, and exporting updated WorkPaper JSON.

Bilig is designed for developers who want formula-backed business logic with explicit readback and testable state, rather than screenshot-based spreadsheet automation.
